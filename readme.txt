== Description ==

Mingle WP-FB AutoConnect Widget

This widget works in conjunction with the WP-FB AutoConnect Plugin to add the facebook login to Mingle's login widget (this would take place of your standard Mingle Login Widget).

Upload and activate.
Under Appearance - Widget find the Mingle-FB Connector
Drag into place where you want it to appear.

YOU MUST INSTALL THE WP-FB AutoConnect for this widget to work from http://wordpress.org/extend/plugins/wp-fb-autoconnect/

== Installation ==

1. Upload the folder Mingle WP-FB AutoConnect.zip to your Wordpress plugin directory.
2. Activate the plugin from the plugin panel.
3. Navigate to the widget configuration panel and add to desired sidebar location.
